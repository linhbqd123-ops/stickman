# Ultimate V2 Shared Assets (All Maps)

You only need to upload once. These files are used on every map.

Place files in:

`assets/ultimates/`

Required filenames:

1. `yasuo_ultimate.png`
2. `saiyan_ultimate.png` (Kamehameha icon)
3. `fpt_ultimate.png`
4. `saitama_ultimate.png`
5. `meteor.png` (FPT meteor sprite)

Example:

```
assets/ultimates/yasuo_ultimate.png
assets/ultimates/saiyan_ultimate.png
assets/ultimates/fpt_ultimate.png
assets/ultimates/saitama_ultimate.png
assets/ultimates/meteor.png
```

Recommended specs:

1. Icon size: 256x256 (minimum 128x128)
2. Meteor size: 96x96 to 160x160
3. Format: PNG with transparency
4. Suggested max size: under 400 KB each

If a file is missing, the game will fall back to procedural visuals for that effect.
