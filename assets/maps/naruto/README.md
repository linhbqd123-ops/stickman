# NARUTO Map Assets

## Background Images
Place in this folder:
- `bg_main.png` — Main background (1600×720)
- `bg_overlay.png` (optional) — Overlay layer for parallax effect

## Platform Images
Place in `./platforms/` folder:
- `plat_ground.png` — Main ground/floor
- `plat_wooden.png` — Wooden ninja platform
- `plat_stone.png` — Rock/stone platform
- `plat_chakra.png` — Glowing chakra platform
- `plat_ninja.png` — Small ninja platform

## Music
Place in `../../music/`:
- `naruto_theme.mp3` — Background music

## Notes
- Backgrounds: 1600×720 pixels
- Platforms: Variable dimensions (check UPGRADE_PLAN.md for specs)
- Format: PNG with transparency support
- Target: < 200KB per image (compress if needed)
