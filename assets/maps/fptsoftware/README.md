# FPT SOFTWARE Map Assets

## Background Images
Place in this folder:
- `bg_main.png` — Main office background (1600×720)
- `bg_dashboard.png` (optional) — Digital dashboard overlay for HUD effect

## Platform Images
Place in `./platforms/` folder:
- `plat_ground.png` — Office floor/carpet
- `plat_desk.png` — Desk/cubicle edge
- `plat_shelf.png` — Bookshelf / server rack
- `plat_server_rack.png` — Server rack (alternative to shelf)
- `plat_monitor.png` — Monitor edge (digital style glow)
- `plat_window.png` — Window frame (neon-lit city outside)

## Music
Place in `../../music/`:
- `fptsoftware_theme.mp3` — Background music (tech vibes)

## Notes
- Backgrounds: 1600×720 pixels
- Platforms: Variable dimensions (check UPGRADE_PLAN.md for specs)
- Format: PNG with transparency support
- Style: FPT Pink (#FF006E), Neon Green (#39FF14), Dark Blue (#0A1E3F)
- Target: < 200KB per image (compress if needed)
